AI Study Assistant

An AI-powered study assistant that helps students transform notes, documents, and YouTube lectures into structured learning material using Google's Gemini API.

Features

- Summarize notes into concise and easy-to-read points
- Explain concepts in simple language
- Generate quizzes from study material
- Extract important key points
- Generate viva and interview questions
- Support for TXT, PDF, and DOCX files
- Analyze YouTube lectures using transcripts
- Web-based interface built with Flask

Supported Inputs

- Manual Notes
- TXT Files
- PDF Files
- DOCX Files
- YouTube Video Links (with available transcripts)

Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Gemini API
- PyPDF2
- python-docx
- youtube-transcript-api
- Git & GitHub

Project Workflow

Input Source → Content Extraction → Prompt Generation → Gemini API → Structured Output

Installation

Clone the Repository

git clone https://github.com/YOUR_USERNAME/AI-Study-Assistant.git

Install Dependencies

pip install -r requirements.txt

Configure Gemini API Key

Add your Gemini API key in the project configuration.

Run the Application

python app.py

Open in Browser

http://127.0.0.1:5000
